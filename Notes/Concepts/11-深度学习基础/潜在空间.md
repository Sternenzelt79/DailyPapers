# 潜在空间

## 定义

潜在空间（Latent Space）是神经网络中间层输出的**高维连续向量空间**，原始输入（token、图像等）在其中以压缩的、语义丰富的向量表示。与离散 token 空间不同，潜在空间是连续的，可进行插值、回归等操作。

## 核心要点

1. **信息压缩**：将高维原始输入压缩为低维连续表示，去除冗余
2. **语义结构**：语义相似的输入在潜在空间中距离更近
3. **可微性**：连续空间支持梯度回传，适合作为回归监督目标
4. **与 token 空间对比**：token 空间是离散的（词表大小维度），潜在空间是连续的（隐层维度）

## 代表工作

- [[NextLat]]: 在潜在空间而非 token 空间进行下一状态预测，学到更紧凑的信念状态
- [[JEPA]] / [[I-JEPA]]: 在潜在空间进行自监督预测，避免像素级重建
