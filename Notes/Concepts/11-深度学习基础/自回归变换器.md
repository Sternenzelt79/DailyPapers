# 自回归变换器

## 定义

基于 Transformer 架构的自回归语言/多模态模型，使用因果（单向）自注意力，通过 next-token 预测方式逐步生成离散序列，是 GPT 系列、VLM 等模型的基础。

## 数学形式

自回归生成目标（最大化条件概率）：

$$
\mathcal{L}_{AR} = -\sum_{i=1}^{T} \log p_\theta(x_i \mid x_1, \ldots, x_{i-1})
$$

因果注意力掩码确保位置 $i$ 只能看到位置 $1, \ldots, i-1$ 的信息：

$$
A_{ij} = \begin{cases} \text{softmax}(\mathbf{q}_i^\top \mathbf{k}_j / \sqrt{d}) & \text{if } j \leq i \\ 0 & \text{otherwise} \end{cases}
$$

## 核心要点

1. **因果性**: 通过 causal mask 保证自回归属性，每个位置只依赖左侧上下文
2. **离散 token**: 标准形式处理离散词汇表 token，VLM 扩展为离散视觉 token
3. **KV-Cache**: 推理时缓存历史 Key/Value，逐 token 高效解码
4. **长上下文**: 现代 VLM 支持 100K+ token 上下文窗口（如 Cosmos 3 的 256K）

## 代表工作

- [[Cosmos3]]: 以自回归变换器为 AR Tower，负责多模态理解和推理（256K token 上下文）
- GPT-4: 大规模自回归语言模型
- Qwen3-VL: 多模态自回归变换器
