# 交叉注意力（Cross-Attention）

## 定义
注意力机制的变体，其中 Query 来自一个序列，而 Key 和 Value 来自另一个序列，实现两个不同来源特征之间的信息交互。

## 数学形式

$$
\text{CrossAttention}(Q, K, V) = \text{softmax}\left(\frac{Q_A K_B^\top}{\sqrt{d_k}}\right) V_B
$$

其中 $Q_A$ 来自序列 $A$，$K_B, V_B$ 来自序列 $B$。

## 核心要点
1. **条件注入**：常用于将文本/状态等条件信号注入到视觉特征流中。
2. **编解码器交互**：Transformer 解码器使用交叉注意力查询编码器输出。
3. 在扩散模型中，视觉特征（Query）通过交叉注意力与文本嵌入（Key/Value）对齐。
4. 与自注意力的区别：自注意力的 Q/K/V 均来自同一序列。

## 代表工作
- [[潜扩散模型]]：文本条件通过交叉注意力注入 U-Net
- [[OrbiSim]]: OrbiSim-Vision 用交叉注意力将对象 token 注入 U-Net 特征，实现物理状态条件渲染
