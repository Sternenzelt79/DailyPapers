# VLA-Cache

## 定义

一种针对 VLA 推理的加速方法，利用机器人操控任务中相邻时步视觉特征的高相似性，通过缓存历史 Token 特征并重用减少重复计算。

## 核心要点

1. **时序连续性利用**: 相邻时步 Token 特征变化缓慢，可缓存复用
2. **特征级缓存**: 不同于 VLA-Pruner 的 Token 选择，VLA-Cache 直接缓存特征向量
3. **效率劣势**: 相比 VLA-Pruner，在等效性能下延迟高 16%、FLOPs 多 20%

## 局限性

- 缓存机制引入额外显存开销
- 高剪枝率下性能退化明显（LIBERO 87.5% 剪枝率下相对精度仅 54.8%）

## 代表工作

- [[VLA-Pruner]]: 与 VLA-Cache 对比，证明 Token 剪枝 + 双重重要性估计优于纯特征缓存方案
