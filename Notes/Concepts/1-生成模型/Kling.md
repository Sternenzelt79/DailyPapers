# Kling

## 定义
快手旗下的视频生成基础模型（Video Foundation Model），支持文本到视频、图像到视频等多种生成模式，以高质量、长时长、运动一致性著称。

## 核心要点
1. 支持静态相机模式，可生成摄像机固定的人-物交互视频
2. 提供 API 接口（Kling API），方便集成到数据生成流水线
3. 2.5 Turbo Pro 版本在生成速度和质量间取得良好平衡

## 在机器人学中的应用
- GRAIL 使用 Kling 2.5 Turbo Pro 在已知3D场景配置下生成人-物交互参考视频
- 静态相机模式确保生成视频与已知相机内外参一致，便于后续运动重建
