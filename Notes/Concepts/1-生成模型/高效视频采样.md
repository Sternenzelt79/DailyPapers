# 高效视频采样

## 定义
一种在视频理解任务中动态减少视频 token 数量的技术，通过跳帧或自适应采样策略降低计算开销，在保持理解质量的同时提升推理效率。

## 核心要点
1. **Token 压缩**: 对视频理解（而非生成）任务选择性减少视频 token 数量
2. **自适应**: 根据视频内容动态调整采样密度（高动态区域保留更多帧）
3. **推理加速**: 减少 Transformer 的序列长度，降低 $O(n^2)$ 注意力计算开销
4. **与生成解耦**: 仅对理解任务做 token 减少，生成任务仍使用完整 token 序列

## 代表工作
- [[Cosmos3]]: 使用 EVS 减少 Reasoner 模式中的视频 token 数量，降低长视频理解的计算成本
