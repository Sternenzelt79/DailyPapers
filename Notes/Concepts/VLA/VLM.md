# VLM (Vision-Language Model)

视觉-语言预训练模型，将图像 / 视频与自然语言对齐到同一个 token 空间。代表家族包括 LLaVA、Qwen-VL、PaliGemma、InternVL 等。在 [[VLA]] 中常作为感知 + 语言理解的 backbone，下游接 action head。

## 代表工作

- LLaVA / Qwen-VL / PaliGemma / [[Qwen3-VL]]
- 几乎所有现代 [[VLA]]（[[Pi05]]、[[GR00T-N1.6]]、[[RLDX-1]]）都以某个 VLM 作为基座。
