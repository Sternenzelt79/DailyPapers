# GAIA-1

## 定义
GAIA-1 是 Wayve 提出的用于自动驾驶的生成式世界模型，能够在给定过去视频帧和动作（方向盘、油门等控制信号）的条件下，生成未来驾驶场景的视频预测。

## 核心要点
1. 将自动驾驶的世界建模问题转化为条件视频生成任务
2. 输入：历史视频帧序列 + 连续控制动作；输出：未来帧视频
3. 展示了视频生成世界模型在具身/自动驾驶领域的早期成功应用
4. 是 2D 视频生成型世界模型路线的代表作之一（另见 Cosmos-Predict）

## 局限性（来自 PNWM 的批评）
- 视觉保真度不等于物理有效性——生成的视频"看起来对"但不遵守物理守恒
- 没有显式的物理先验，长时序稳定性依赖数据分布而非物理约束
