# RISE

## 定义
一种强化学习训练框架，通过迭代自增强机制训练多模态或 Agent 模型，常用于需要多轮规划和执行的任务。

## 核心要点
1. 基于自生成数据的迭代训练
2. 适用于多模态 Agent 的序列决策任务
3. 与 [[GRPO]] 类方法结合使用
