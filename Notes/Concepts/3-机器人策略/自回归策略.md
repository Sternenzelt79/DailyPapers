# 自回归策略

## 定义
自回归策略（Autoregressive Policy）是一种将机器人动作离散化为 Token 序列，并使用自回归语言模型逐 Token 预测的策略学习方法，将动作预测统一为序列生成问题。

## 核心要点
1. **离散化**：连续动作通过量化（如 VQ-VAE 或均匀分箱）转化为离散 Token
2. **自回归生成**：逐 Token 生成动作序列，每个 Token 的预测条件依赖之前所有 Token
3. **与 LLM 统一**：可直接在预训练 LLM 上微调，利用语言先验
4. **局限**：推理延迟较高（逐 Token 解码），对高频控制挑战较大

## 代表工作
- [[OpenVLA]]: 基于 LLaVA 的自回归 VLA，将动作预测为离散 Token 序列
- [[RT-2]]: Google 的自回归 VLA，将机器人动作映射为文本 Token
- [[VLA-Pruner]]: 验证了 Token 剪枝对 OpenVLA（自回归）的加速效果
