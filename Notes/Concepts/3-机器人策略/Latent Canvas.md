# Latent Canvas

## 定义
NavWAM 提出的结构化帧序列设计，将机器人状态、目标图像、当前观测、动作块、未来状态、未来观测和目标进度值统一编排为九帧视频格式，供扩散-Transformer 骨干进行联合去噪。

## 核心要点
1. 将所有模态（RGB、数值、标量）统一编码为图像格式，利用[[因果 VAE]]压缩为潜变量
2. 帧 0–3 为条件帧（已知），帧 4–8 为生成帧（预测目标）
3. 数值型帧（状态、动作、价值）通过像素平铺（pixel tiling）实现模态对齐
4. 支持三种条件化训练模式：Policy / World-Model / Value，推理时灵活切换

## 帧结构

| 帧 | 内容 | 角色 |
|----|------|------|
| 0 | 空白 padding | 条件 |
| 1 | 机器人状态 $[x,y,\psi]$ | 条件 |
| 2 | 目标图像 $g$ | 条件 |
| 3 | 当前观测 $o_t$ | 条件 |
| 4 | 动作块 $a_{t:t+H-1}$ | 生成 |
| 5 | 未来状态 $s_{t+H}$ | 生成 |
| 6 | 未来观测 $o_{t+H-1}$ | 生成 |
| 7 | 未来观测 $o_{t+H}$ | 生成 |
| 8 | 目标进度值 $v_{t+H}$ | 生成 |
