### 1. WBench: A Comprehensive Multi-turn Benchmark for Interactive Video World Model Evaluation
- **作者**: Kaining Ying et al.
- **机构**: 复旦大学，美团 Longcat Team
- **链接**: [arXiv](https://arxiv.org/abs/2605.25874) | [PDF](https://arxiv.org/pdf/2605.25874)
- **来源**: 🔥 HF Trending，⬆️ 69

![](https://arxiv.org/html/2605.25874v1/x1.png)

- **核心方法**: WBench 定义了交互式视频世界模型的五维评测框架：视频质量、场景一致性（setting adherence）、交互一致性（interaction adherence）、时序连贯性、物理合规性。数据集包含 289 个测试用例、1058 轮交互。每个用例指定初始场景 + 动作序列，模型需要生成相应视频响应。评测涵盖 [[Matrix-Game]]、LongCat、YUME、Doubao-Seed 等多款商业/开源世界模型。
- **对比方法/Baselines**: 对比 [[Matrix-Game]]、LingBot、Genie 3 等，借鉴了 FVD、CLIP 等视频/语义评估指标
- **借鉴意义**: 如果你要评测自己的 action-conditioned video generation 模型，WBench 是目前最系统的多轮交互评测方案。289 个 case 覆盖导航、物体操作等多类场景，比 FVD 这种无人机指标靠谱得多。
- **锐评**: 补了一个真实存在的空白——现有世界模型 benchmark 要么只测单步质量，要么不管动作一致性。五维框架的设计思路清晰，对 interaction adherence 的单独建模有价值。但 289 个 case 体量较小，且主要评测游戏/室内导航场景，机器人操作覆盖不足。物理合规性的评测靠 VLM judge，可靠性有待验证。 👀
- **关联笔记**: [[Matrix-Game]] — 被测对象之一；[[Diffusion Model]] — 多数被测模型基于视频扩散

### 2. Drift-Resistant Navigation World Model with Anchored Epipolar Guidance
- **作者**: Po-Chien Luan, Zimin Xia
- **机构**: EPFL VITA Lab
- **链接**: [arXiv](https://arxiv.org/abs/2605.24761) | [PDF](https://arxiv.org/pdf/2605.24761)
- **来源**: 📄 arXiv 关键词检索

![](https://arxiv.org/html/2605.24761v1/x1.png)

- **核心方法**: DR-NWM 针对自回归导航世界模型的两类漂移——感知漂移（噪声累积）和几何漂移（预测轨迹偏离运动几何）——提出双向 epipolar masking + anchor 条件 Diffusion Transformer。在每次 rollout 中引入 anchor 帧（过去真实观测），通过 bidirectional epipolar attention 约束生成帧必须满足多视图几何一致性，输入输出均为 ego-view 图像序列 + 动作。
- **对比方法/Baselines**: 对比 [[NWM]]（基础导航世界模型）、EgoWM，在 SCAND、HuRoN、TartanDrive 数据集上测试；借鉴 [[DreamSim]] 做感知质量评估；规划阶段使用 [[Cross-Entropy Method]] (CEM)
- **借鉴意义**: 如果你在做基于视频世界模型的 robot navigation 或 embodied planning，epipolar 约束这个思路可以直接借。比强行用 depth prior 优雅得多，且不需要额外深度标注。
- **锐评**: 方向对，问题定义清楚（双类漂移分开解决）。epipolar masking 是合理的几何约束，但仅在 sim/室内导航数据上验证（无 `has_real_world` 真机数据），对 dynamic object 的处理摘要未提及。CEM 规划在高维动作空间里效率还是个问题，作者也提到这是 limitation。 👀

### 7. X-DiffVLA: X-Embodied Diffusion Action Heads for Vision-Language-Action Models
- **作者**: Boyu Li et al.
- **机构**: 中科院自动化所，BAAI，BeingBeyond
- **链接**: [arXiv](https://arxiv.org/abs/2605.25044) | [PDF](https://arxiv.org/pdf/2605.25044)
- **来源**: 📄 arXiv 关键词检索

![](https://arxiv.org/html/2605.25044v1/x1.png)

- **核心方法**: X-DiffVLA 解决跨具身 VLA 的泛化问题。核心有三：(1) **Embodied Forcing (EBF)**——把具身形态信息（DoF、关节配置）编码为 soft prompt 注入 VLM，不做具身特定 fine-tuning；(2) **Morphological Tree Diffusion (MCTD)**——把不同机器人的运动学树结构编码进扩散过程，用 Monte Carlo Tree 对动作空间做结构化采样；(3) 扩散 action head 替代 autoregressive tokenization，直接输出连续动作分布。在 RoboCasa、XR-1 真机上验证，`has_real_world` = True。
- **对比方法/Baselines**: 对比 [[OpenVLA]]、UniVLA、DemoGrasp 等；借鉴 [[FAST]] 做 action tokenization 参考；真机用 XR-1 双臂机器人
- **借鉴意义**: EBF 的 soft prompt 具身编码方案直接可用——如果你要做跨机器人策略迁移，这比 domain-specific fine-tuning 高效。MCTD 在多峰分布动作上的处理值得关注。
- **锐评**: 这篇有真货。跨具身泛化是 VLA 的核心难题，EBF+MCTD 的组合比以往"一个模型微调多次"的暴力方案优雅。消融实验章节存在（IV-E），真机 demo 在 IV-D。但有一个问题：MCTD 在高 DoF 机器人上的计算开销摘要未给出数字，推断复杂度可能不低。Morphological tree 的构建需要先验运动学知识，对未知具身的泛化能力需要看全文。 🔥

### 8. QuoVLA: Quotient Space for Vision-Language-Action Models
- **作者**: Wang, Xuan, Wu, Yinan, Duan, Haoran, Han, Jungong
- **机构**: 清华大学自动化系
- **链接**: [arXiv](https://arxiv.org/abs/2605.24890) | [PDF](https://arxiv.org/pdf/2605.24890)
- **来源**: 📄 arXiv 关键词检索

![](https://arxiv.org/html/2605.24890v1/o.jpg)

- **核心方法**: QuoVLA 用商空间（Quotient Space）理论重新审视 VLM 到 VLA 的 representation。核心主张：预训练 VLM 的 latent 并非对 action 不够用（action-insufficiency），而是包含了可以直接利用的 action 信息——问题在于如何正确"提取商"。提出 Transformer-based Prefix Quantization 做 minimal VLM-action quotient，+ Dual-Branch Action Generation（一支生成离散 token，一支生成连续 action）。在 LIBERO、RoboTwin 2.0 上验证，但 `has_real_world` = False——无真机实验。
- **对比方法/Baselines**: 对比 [[OpenVLA]]、UniVLA、SimVLA、[[SpatialVLA]]、[[ACT]]、[[OpenVLA-OFT]] 等；在 [[LIBERO]]、RoboTwin 上测试
- **借鉴意义**: 理论框架有意思——如果 VLM latent 本来就含 action 信息，那么现有 VLA 的设计可能在"去掉"有用信息。Dual-branch 生成对需要同时预测离散指令和连续动作的任务有参考价值。
- **锐评**: 商空间理论的引入有学术新意，比"我们加了个新 loss"更有原理支撑。但没有真机实验是硬伤——LIBERO 的 sim 成绩说服力有限。Dual-branch 设计是否真的比 single head 好，还是只是参数量更大带来的提升，需要严格消融。 👀

### 9. Afford-VLA: Action-Aligned Visual Planning via Internalized Affordance
- **作者**: Runze Wang et al.
- **机构**: 复旦大学，KAUST，上海交大，华东师大
- **链接**: [arXiv](https://arxiv.org/abs/2605.24203) | [PDF](https://arxiv.org/pdf/2605.24203)
- **来源**: 📄 arXiv 关键词检索

![](https://arxiv.org/html/2605.24203v1/x1.png)

- **核心方法**: Afford-VLA 把可供性（affordance）内化为 VLA 自身的视觉规划能力，不依赖外部可供性检测器。输入图像 + 指令，模型先生成 **Internal Affordance Mask**（标注应该在哪里操作），再基于 mask 做 **Affordance-Conditioned Action Prediction**。关键是 Action-Aligned Training：可供性 mask 和 action 联合监督，保证 mask 的区域真的与动作方向对齐（而不只是语义区域）。有 LIBERO + SimplerEnv 仿真和真机实验。
- **对比方法/Baselines**: 对比 [[OpenVLA]]、[[SpatialVLA]]、[[OpenVLA-OFT]]；借鉴 [[SimplerEnv]] 作为评测环境；真机用自建 real-world benchmark
- **借鉴意义**: 内化可供性的思路可以减少对外部视觉模块的依赖，对做 end-to-end 操作的研究者有直接价值。Action-aligned training 的 loss 设计可以复用。
- **锐评**: 问题定位准确——现有 VLA 在"在哪里操作"上确实弱。但 affordance mask 的生成本身需要标注（需要 joint affordance & action dataset），数据收集成本不低，这可能限制 scalability。真机实验在 Appendix E，不是主表。 👀

### 10. OASIS: Observation-Action Space Alignment via SE(3) Trajectory Prediction for Robotic Manipulation
- **作者**: Xinzhe Chen, Sihua Ren et al.
- **机构**: 西安交通大学（人机混合增强智能国家重点实验室）
- **链接**: [arXiv](https://arxiv.org/abs/2605.25829) | [PDF](https://arxiv.org/pdf/2605.25829)
- **来源**: 📄 arXiv 关键词检索

![](https://arxiv.org/html/2605.25829v1/x1.png)

- **核心方法**: OASIS 的核心论点：VLA 和 WAM 的中间表示停留在 observation space，但 action space 是刚体几何（SE(3)），强迫 action decoder 从 observation-space 表示中隐式恢复几何是不高效的。OASIS 提出用 SE(3) 轨迹预测作为对齐中间表示，3D-aware feature encoder 提取空间特征，SE(3) Trajectory Predictor 显式预测末端执行器的 6-DOF 轨迹，closed-form action decoder 用几何解析解而非神经网络生成最终动作。有 LIBERO、CALVIN 仿真和真机实验。
- **对比方法/Baselines**: 对比 [[ACT]]、[[RDT]]、AuxTraj、Seer-Large；借鉴 [[WAM]] 框架；对比 ThinkAct；基于 LIBERO/CALVIN/ABC-D benchmark
- **借鉴意义**: SE(3) 对齐表示的思路对做 manipulation 的人很有价值——显式建模刚体约束比端到端隐式学习更有几何可解释性。closed-form decoder 也可以大幅降低推理延迟。
- **锐评**: 理论动机充分，几何先验注入是合理方向。但 SE(3) trajectory predictor 的训练需要高质量的 6-DOF 轨迹标注，在 wild 数据上的 scalability 存疑。真机实验在 Appendix，而非主表，需要看细节。与 [[WAM]] 系列工作的边界不够清晰。 👀

- **核心方法**: VLA-Pruner 针对"语义-动作 gap"——VLM 的 token importance 是基于语义对齐，但 VLA 的 action decoder 关注的 token 分布不同。核心发现：action token 对视觉 token 的 attention 在时序上有连续性（Temporal Continuity of Action-to-Vision Attention），基于此设计 stage-wise token pruning：prefilling 阶段静态剪枝 + decoding 阶段动态更新。有 [[OpenVLA]] 和 [[pi0]] 在 LIBERO、SimplerEnv、SIMPLER 上的真机对比。
- **对比方法/Baselines**: 对比 FastV、SparseVLM、DivPrune、EfficientVLA；基于 [[OpenVLA]] 和 [[OpenVLA-OFT]] 实验；有 SIMPLER 真机平台验证
- **借鉴意义**: 这是第一篇专门分析 VLA 的 token importance 分布和 VLM 差异的工作。Temporal Continuity 的 insight 可以用于任何需要实时推理的 VLA 系统。
- **锐评**: 分析很扎实，发现确实有价值。但 v4 版本说明作者已经反复修改，原始版本（2511.16449）可能在 reviewer 那里过了好几轮。真机验证的任务类型和成功率需要仔细对比——pruning 后成功率有没有显著下降？ 👀

### 12. EXPO-FT: Sample-Efficient Reinforcement Learning Finetuning for Vision-Language-Action Models
- **作者**: Perry Dong, Kuo-Han Hung
- **机构**: Stanford University

## 📈 今日趋势判断

VLA 方向已经进入"精细化内卷"阶段——跨具身（X-DiffVLA）、token 剪枝（VLA-Pruner、SpecPrune）、RL 微调（EXPO-FT）、几何注入（OASIS、GeoVLA）各占山头，都是在已有框架上做增量改进。真正有范式意义的工作是 LIBERO-PRO：它告诉我们现有 benchmark 已经失效，整个社区的"95% 成功率"可能是幻觉。World model 方向比 VLA 更早期，NanoWM 的极简框架填了一个工具缺口，但缺乏接近 OpenAI/Google 量级的基础模型支撑。今天最值得追的还是 X-DiffVLA + LIBERO-PRO 这对组合：一个做更好的 VLA，一个告诉你怎么更诚实地评估它。
