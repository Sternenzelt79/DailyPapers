### 1. LongCat-Video Technical Report

- **作者**: Meituan LongCat Team
- **机构**: 美团
- **链接**: [arXiv](https://arxiv.org/abs/2510.22200) | [PDF](https://arxiv.org/pdf/2510.22200)
- **来源**: 🔥 HF Trending，⬆️ 35

![](https://arxiv.org/html/2510.22200/x1.png)

- **核心方法**: LongCat-Video 是一个 13.6B 参数的视频生成模型，主打高效长视频推理。架构上采用 [[DiT]] 基座，结合 [[Video-Continuation]] 机制实现长视频自回归生成，训练中引入多奖励 [[RLHF]] 和 [[GRPO]] 进行偏好对齐。重点卖点是"multi-shot"长视频：支持跨场景、跨镜头的连贯生成，区别于单镜头外推的方式。数据管线（preprocessing + quality filtering）是质量的核心支撑，section headers 显示有专门的 Data Curation Pipeline 章节。
- **对比方法/Baselines**: 对比 [[PixVerse]]；借鉴 [[SDE]] 采样框架。
- **借鉴意义**: 视频生成向世界模型迁移的路上，大参数+RLHF+multi-shot 连贯生成是这篇的核心贡献。对做 world model 的人值得看数据管线设计和长视频对齐策略——GRPO 用于视频生成对齐还比较少见。
- **锐评**: 13.6B 参数但是技术报告，不是正式论文，所以方法细节披露程度有限。"critical pathway toward world models"这句话说得很漂亮，但视频生成真的就是世界模型了吗——物理一致性、可控性、行动条件生成全没提。RLHF 视频对齐打磨视觉质量没问题，但跟 embodied AI 里需要的预测准确性是两回事。美团出的模型，工程能力毋庸置疑，学术 novelty 存疑。👀

## Diffusion / 生成模型

### 4. SANA-Video: Efficient Video Generation with Block Linear Diffusion Transformer

- **作者**: Junsong Chen, Yuyang Zhao, Jincheng Yu 等
- **机构**: NVIDIA
- **链接**: [arXiv](https://arxiv.org/abs/2509.24695) | [PDF](https://arxiv.org/pdf/2509.24695)
- **来源**: 🔥 HF Trending，⬆️ 53

> ⏪ **再推提醒**：这篇在 2026-05-16 推荐过

- **核心方法**: Block Linear [[DiT]] 把 softmax attention 替换为线性 attention 做 block-level 处理，配合 [[DCAE]] 高压缩比编码器降低 token 数量；自回归长视频通过块级 chain-rule 分解生成。可在 RTX 5090 部署，720×1280 分辨率，分钟级时长。
- **锐评**: 效率驱动的视频扩散论文，架构创新点（block linear attention + DCAE）组合有一定新意。但对做 robotics world model 的人来说，关注点在生成逼真度和文本对齐，而不是 embodied 场景的物理一致性——这块没提。可部署性不错，但缺少 action-conditioned 生成。🫠

- **作者**: Zhennan Chen
- **机构**: Nanjing University、Tencent
- **链接**: [arXiv](https://arxiv.org/abs/2605.12013) | [PDF](https://arxiv.org/pdf/2605.12013)
- **来源**: 🔥 HF Trending，⬆️ 30

> ⏪ **再推提醒**：这篇在 2026-05-13 推荐过

- **核心方法**: Latent-to-Pixel（L2P）迁移范式——把预训练 [[LDM]] 的知识直接迁移到像素空间扩散模型，避免从头训练像素级模型的巨大计算开销。核心是用潜在空间预训练权重作为初始化，再在像素空间 fine-tune。

## 架构 / 基础

### 8. Gated DeltaNet-2: Decoupling Erase and Write in Linear Attention

- **作者**: Ali Hatamizadeh, Yejin Choi, Jan Kautz
- **机构**: NVIDIA
- **链接**: [arXiv](https://arxiv.org/abs/2605.22791) | [PDF](https://arxiv.org/pdf/2605.22791)
- **来源**: 🔥 HF Trending，⬆️ 20

- **核心方法**: 对 [[KDA]]（Kimi Delta Attention）做改进，把 erase 和 write 两个操作解耦，分别由独立门控 $\bm{g}$（erase gate）和值路径（write gate）控制。核心公式：$\bm{G}_r = \sum_{i=1}^r \bm{g}_i$，$\bm{\gamma}_r = \exp(\bm{G}_r)$，在 chunkwise 计算下保持与 KDA 相同的结构。实验覆盖 NIAH-1/2/3 长上下文任务、[[RULER]]、[[ARC]]、[[LAMBADA]] 等。
- **对比方法/Baselines**: 与 [[KDA]] 直接对比；参考 [[WY]]（Woodbury-Hadamard representation）；在 [[NIAH]] 系列 benchmark 和 [[ARC]]、[[PIQA]]、[[BoolQ]] 上评测。
- **借鉴意义**: 线性 attention 对做长序列 policy 有潜力——推理时固定大小 recurrent state 意味着可以做低延迟推理。但这篇是纯 NLP 架构论文，没有任何 robotics 实验，需要自己去做迁移。
- **锐评**: 从 KDA 出发做改进，erase/write 解耦的动机清晰，数学推导严谨。但 upvotes 只有 20，说明社区评价也一般。NVIDIA 三位作者，工程质量有保障，但理论 novelty 有限。对 embodied AI 的价值是间接的，先在 NLP 上验证了机制，距离 robot policy 应用还有一段路。🫠
