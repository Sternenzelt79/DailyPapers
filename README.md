# Daily Paper Skills

A set of [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skills that automatically fetch, score, review, and generate notes for daily AI/ML papers — tailored for researchers in **Embodied AI, World Models, and Diffusion Models**.

Papers are sourced from arXiv and HuggingFace, scored by keyword relevance, enriched with metadata, then reviewed with opinionated commentary and saved as Obsidian-compatible markdown.

## Reference

This project uses Claude Code skills from: https://github.com/huangkiki/dailypaper-skills
